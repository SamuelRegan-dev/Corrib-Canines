# Corrib Canines

Website for a local dog walking and sitting business based in Galway.

(https://corribcanines.netlify.app/)
## Tech Stack
- HTML/CSS/JS
- Netlify Functions (serverless)
- Nodemailer
- reCAPTCHA v3

## Features
- Contact form with server-side email handling
- Rate limiting and bot protection
- Security headers

## Setup
Clone the repo and create a `.env` file with the following variables:
EMAIL_USER=
EMAIL_PASS=
RECAPTCHA_SECRET=
